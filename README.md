# 42fract_ol
This project is about creating graphically beautiful fractals.

## Compiling and running
Run `make`. An executable will compile.

## usage
`./fractol [fractal]`

## Fractals
- Mandelbrot
- Julia
- Burning Ship

## Controls
- Scroll to zoom in and out
- `arrow keys` to move view
- `space` to reset view color
- `esc` to exit
